# Contributing

This project implements security-sensitive infrastructure. 
While aPlane is in alpha, external pull requests are not yet accepted.

The code is published as open source for transparency, independent
audit, and reuse under the terms of the license. Forks are explicitly
permitted.

### Security Issues

If you believe you have identified a security vulnerability, please
open an issue with a minimal reproduction or description, or email
makman.algo@gmail.com for private disclosure. Do not include private
keys or sensitive material.

All reports are handled at the maintainer's discretion.
